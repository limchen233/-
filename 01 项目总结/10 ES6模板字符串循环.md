最近使用`ES6`的模板字符串动态渲染`html`，然后循环数据，可以这样写：

```jsx
if (item.data) {
  var temp = `${item.data.map(items => { 
    return `<input type="checkbox" value="${items.value}" lay-filter="filter" 
               lay-skin="primary" title="${items.label}" />
            </br>`
  }).join('')}`
  return temp
}
```
